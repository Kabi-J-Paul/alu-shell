#Processes and Signals
Bash scirpts for working wiht processesa nd signals on Linux
##Tasks
- 0 — displays the script's own PID
- 1 — lists all running processes with hierarchy
- 2 — shows lines containing "bash" from process list
- 3 — shows bash PIDs using pgrep
- 4 — prints a message indefinitely
- 5 — stops the infinity process using kill
- 6 — stops the infinity process without kill
